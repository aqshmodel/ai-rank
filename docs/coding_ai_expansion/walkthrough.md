# AIコーディングエージェント・ツール追加 ウォークスルー

2026年における最新のAI開発ツールのトレンドを反映するため、THE AI RANKの用語集に対し、全27件の新規ツールの追加とビジネス・マネジメント層向け解説（ROI視点）の付与を実行しました。

## 完了した作業内容

1. **既存用語との重複調査**: 
   ご提示いただいた全ツールリストから、既存用語（Cursor、Claude Code等）と重複するものを除外・統合し、新規に単独で作成すべき「27件」のコア・ツール群を抽出しました。

2. **6つのバッチ分割と一括作成**:
   27件を6つのカテゴリ（バッチ）に分類し、すべての項目について最新のWebリサーチ（2026年4月現在）を行い、The AI Rank特有のフォーマット（フロントマター、技術機能、ビジネスROI視点）にてMarkdownファイルを新規作成しました。

### 追加された27件のリソース一覧

#### 1. Web UI生成 ＆ Claude拡張系
- **Lovable (`lovable.md`)**: MVP構築の壁を突破するSupabase連携フロントエンド特化AI。
- **Claude Projects ＋ Code (`claude-projects.md`)**: 戦略と実行を分割・保持するスプリットブレイン開発。
- **Claude for Excel (`claude-for-excel.md`)**: スプレッドシート業務を自動化するバックオフィスROI。
- **Claude for PowerPoint (`claude-for-powerpoint.md`)**: 他オフィスソフトの文脈を引き継ぐ資料作成ゼロ化ツール。
- **Claude in Chrome (`claude-in-chrome.md`)**: 既存SaaS・CRM間の転記作業を全自動化するブラウザエージェント。

#### 2. Googleインフラ ＆ 次世代クラウド・エディタ系
- **Gemini CLI (`gemini-cli.md`)**: 200万トークン対応、巨大システムの全容リバースエンジニアリング。
- **Gemini Code Assist (`gemini-code-assist.md`)**: エンタープライズ特化のセキュリティと社内コード対応エージェント。
- **Windsurf (`windsurf.md`)**: 予測によるスイッチングコスト排除（フロー状態維持）特化IDE。
- **Zed AI (`zed-ai.md`)**: Rust製・超低遅延で開発者の認知疲労を防ぐ爆速エディタ。
- **Replit (`replit.md`)**: プロンプトのみで環境構築からデプロイまで完結するクラウドIDE。

#### 3. ターミナル・自律エージェント特化
- **Warp (`warp.md`)**: インフラ系運用スキルをAI化し、Linuxコマンドの属人化を防ぐAIターミナル。
- **Cline (`cline.md`)**: 透明性・Human-in-the-loopに特化し、破壊的変更リスクを下げる自律エージェント。
- **Roo Code (`roo-code.md`)**: ArchitectやCoderなど専門モードを切り替え、個人の多能工化を実現する派生エージェント。
- **Aider (`aider.md`)**: 変更をGitアトミックコミット化し、失敗時のロールバックを保証するターミナルAI。

#### 4. エンタープライズ・大規模コード解析特化
- **Augment Code (`augment-code.md`)**: 数百万行のレガシー解読に対応するエンタープライズ・AIアシスタント。
- **Sourcegraph Cody (`sourcegraph-cody.md`)**: 検索グラフとRAGを活用し、全社のナレッジサイロ化を解消。
- **Amazon Q Dev (`amazon-q-dev.md`)**: AWS基盤最適化とレガシーのクラウド移行（モダナイズ）自動化。
- **Tabnine (`tabnine.md`)**: 情報漏洩リスクゼロ（エアギャップ対応）の最強プライバシー・ローカルAI。
- **JetBrains AI (`jetbrains-ai.md`)**: IDEの静的解析基盤を活かし、安全なリファクタリング（品質保証）を実現。

#### 5. ローカルLLM・非依存エンジン系
- **Qwen Code (`qwen-code.md`)**: ローカルでも動く最強の推論モデル。クラウド費用の削減。
- **iFlow (`iflow.md`)**: AIツールの短命なライフサイクルと、ベンダーロックイン回避のケーススタディ。
- **Continue (`continue.md`)**: APIキーもローカルモデルも切り替え自在（BYOK）なオープンリソースの王。
- **Kilo Code (`kilo-code.md`)**: 500以上のモデルに即座に乗り換え可能な、究極のアグノスティック・エージェント。

#### 6. 爆速補完・ナレッジ管理系
- **Supermaven (`supermaven.md`)**: 100万コンテキストを超低遅延（待機時間ゼロ）で返し、タイピング効率を極限化。
- **Pieces (`pieces.md`)**: ローカルファーストでコード履歴を「第二の脳」として蓄積・シェアするナレッジ管理。
- **Phind (`phind.md`)**: 公式ドキュメントへのリアルタイム検索に特化し、AIの「嘘（ハルシネーション）」調査工数を撲滅。
- **Codeium (`codeium.md`)**: 既存IDEにゼロコストで組み込め、個人から法人まで段階的なAI投資が可能な汎用エンジン。
