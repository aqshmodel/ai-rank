# ユーザーインサイトマップ（The AI Rank・地方企業向け）

## ターゲットペルソナ
**岩手県および全国の地方中小企業の経営者・IT/DX担当者**
- 年齢層: 40代〜60代の経営層 / 30代の若手担当者
- リテラシー: 「DXやAIをやらないとまずい」とは思っているが、何から手をつければいいか分からない。
- ペイン（痛み）: 
  - ベンダーに数百万円払ってシステムを入れたが誰も使わない。
  - AIコンサルに依頼したら「高額なレポート」だけ出てきて現場が変わらなかった。
  - プロンプトという言葉すら社内に浸透していない。

## 彼らが検索する「隠れた欲求」
### 表面的な検索意図（顕在）
- 「AI 導入 失敗 事例」
- 「地方企業 DX 助成金」
- 「ChatGPT 社内研修」

### 深層の欲求（潜在）
- 「もうITツールで失敗したくない」
- 「現場の泥臭い運用まで付き合ってくれるパートナーが欲しい」
- 「うちの古い業界・特殊な業務でもAIは使えるのか知りたい」

## コンテンツによる対応策
The AI Rankのブログでは、SaaSの機能比較のような**「キラキラしたIT記事」は一切書かない**。
代わりに、「高額コンサルに騙された事例」や「とりあえずChatGPTを入れた企業が辿る末路」など、**生々しく泥臭いリアルな共感（Pain）** を提示し、そこから「自社データを入れた専用AIエージェントの開発と、現場への伴走支援（Aqshの強み）」へと誘導する。
