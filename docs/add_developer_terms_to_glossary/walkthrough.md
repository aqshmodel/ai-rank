## 留意した点（AI活用時代の切り口）
すべての用語に対して、単なるIT用語としての解説だけでなく、**「AI導入・活用にどう関わるか（注意点）」**というAI Rank特有のアプローチを実装しました。

**【主な切り口の例】**
- **.env / .gitignore / 秘密鍵**:
  AI拡張機能を利用した開発を行う際、これらを読み込み除外にしないと、AI側にパスワードや鍵が送信されて漏洩する致命的なリスクがあることを強く警告。
- **Docker / 開発コンテナ**:
  AIエージェントに自動でコードを実行させる際、「最悪壊されてもよい箱」として切り離したコンテナ環境を用いることの必須性。
- **rm / SSH / 権限制御**:
  AIに本番サーバーの権限を与えてしまうと、`rm -rf` 等によって不可逆な破壊をもたらすことの恐ろしさと安全な運用。
