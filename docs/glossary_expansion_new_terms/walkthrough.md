# 用語集追加完了 ウォークスルー (Google Vids, OpenCraw, Claude Mythos, Munus)

このドキュメントは、指定された4つの最新AI関連用語を The AI Rank いわて の用語集に追加した作業の概要と検証結果をまとめたものです。

## 実施した変更

新規で以下のマークダウンファイル4件を `terms` ディレクトリ内に作成し、用語追加を完了しました。各ファイルには既存フォーマットに基づくFrontmatter（タイトル、説明、日付、ふりがな、カテゴリ）を付与し、さらに「ビジネス・ROI観点での影響」という当サイトの特色となる深い洞察を盛り込んでいます。

1. **`terms/google-vids.md`**： `Google Vids`
   - **概要**：Google Workspace上でブラウザ完結するAI動画生成・編集ツール。
   - **ROIの焦点**：「テキスト中心の非効率なコミュニケーション」から「動画ベースの超効率的なナレッジ共有」へのシフトによる、コンテンツ制作の手間と外注費用の劇的な削減。

2. **`terms/opencraw.md`**： `OpenCraw`
   - **概要**：ブラウザ操作やアプリ操作を人間に代わって自律的に実行するローカルで稼働するパーソナルAIエージェント。
   - **ROIの焦点**：全社規模のRPAではなく、個人の環境にプロキシ型で常駐することで、反復的なWeb調査・フォーム入力を代行する「個の生産性革命」と工数削減。

3. **`terms/claude-mythos.md`**： `Claude Mythos`
   - **概要**：Anthropic社が特定の防衛的コンソーシアム「Project Glasswing」に向けて提供しているサイバーセキュリティ特化の自律型フロンティアLLM。ゼロデイ脆弱性の発見等に対応。
   - **ROIの焦点**：常に不足するセキュリティエンジニアの代役となり、インシデント未然防止による億単位のリスクを無効化する防御的AI投資としての価値。

4. **`terms/munus.md`**： `Munus`（マナス）
   - **概要**：PCのワークフロー（ブラウザやExcelなど様々な外部サービスを跨ぐ操作）を目的から逆算して自律実行する次世代の汎用AIエージェント。
   - **ROIの焦点**：プロセス作業そのものの完全自動化。マネージャーが「AI部下」に作業を丸投げし、人間は意思決定とプランニングにのみ集中できる究極の人件費最適化環境の実現。

## 検証結果

- **書式の整合性**: すべてのファイルに対して、Markdown構文エラーがないこと、および既存の `[用語](/terms/...)` 記法やFrontmatterフォーマット（`category: "AIモデル・ツール"` 等）との一貫性があることを確認しました。
- **反映の確認**: 当該プロジェクトはNode.js/Express環境にて `marked` および `gray-matter` を使用して動的にMarkdownファイルをレンダリングするアーキテクチャであるため、静的なビルドプロセス（`npm run build`）は不要であり、ファイル保存と同時に反映される構造であることを確認・検証済みです。
