# 著者プロフィールマスタ (E-E-A-T対応)

The AI Rank ブログシステムでは、SEOにおけるE-E-A-T（専門性・権威性・信頼性・経験）を担保するため、記事の下部に著者プロフィールを出力します。

## 著者の一覧と設定方法
各記事のフロントマター（`.md` ファイルの上部）に `author: "[ID]"` を指定することで、システム側で以下の情報が自動的にレンダリングされます。
