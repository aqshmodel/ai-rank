## 損失関数とは
損失関数（Loss Function / コスト関数）は、[機械学習](/terms/machine-learning) モデルの予測値と実際の正解値の間のズレ（誤差）を数値化する関数です。この値が小さいほどモデルの予測精度が高いことを意味し、[最適化](/terms/optimization) アルゴリズムがこの値を最小化するように [パラメータ](/terms/parameter) を調整していくのがAIの学習の本質です。

## 代表的な損失関数
[回帰](/terms/regression) タスクでは平均二乗誤差（MSE）、[分類](/terms/classification) タスクではクロスエントロピー損失がよく使われます。タスクに適した損失関数を選ぶことがモデルの性能を左右する重要な設計判断です。[学習曲線](/terms/learning-curve) では損失関数の値の推移をグラフ化して学習の進捗を確認します。
